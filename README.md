# ebay-flask-app

Install ebaysdk

Install flask

Install Beautiful soup

Navigate to the ebay-flask-app folder in your terminal or command prompt then enter


FLASK_APP=search.py flask run


to start your server then on your web browser enter


http://127.0.0.1:5000/


to run the file.
